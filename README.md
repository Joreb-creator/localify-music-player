# Localify Music Player

A Spotify-inspired local music player built with React, TypeScript, Vite, IndexedDB, and the Web Audio API.

## Features

- Local MP3 upload
- Drag-and-drop upload
- IndexedDB local storage
- Artwork extraction
- Queue system
- Shuffle & repeat
- Audio visualizer
- Playlists
- PWA support
- Offline-ready

## Tech Stack

- React
- TypeScript
- Vite
- IndexedDB
- Web Audio API
